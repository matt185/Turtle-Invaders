<h3> Project setup </h3>
 
Create the virtual environment and install the requirements on mac:
<br/>
```
python3 -m venv venv
source venv/bin/activate
python install -r requirements.txt
```
<br/>
After that run the following script for run the code:


```
python main.py
```
<br/>
and enjoy.
